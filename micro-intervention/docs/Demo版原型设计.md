# 微干预 Demo版 原型设计

> 版本：v1.0
> 日期：2026-05-12
> 设计原则：温暖直观，像支持型伙伴

---

## 一、设计理念

### 1.1 核心主张

> **让每一个情境问题，都能转化成为一次有效干预**

### 1.2 页面目标

每个页面只做一件事，目标是清晰的。

### 1.3 流程极简

```
输入情境 → 获得方案 → 执行训练 → 感知进步
   ↓           ↓           ↓           ↓
  描述问题    查看目标    按步执行    看到成长
```

---

## 二、页面结构（5个页面）


## 四、完整用户旅程

```
┌─────────────────────────────────────────────────────────────┐
│                      用户旅程                              │
└─────────────────────────────────────────────────────────────┘

[打开App] → [首页：描述问题/选择场景]
              ↓
        [AI匹配方案]
              ↓
        [方案确认页：了解目标，开始训练]
              ↓
        [今日训练：按步骤执行]
              ↓
        [反馈页：今天表现如何]
              ↓
        [进步展示：看到成长]
              ↓
        [返回首页：查看进度，继续坚持]

关键停留点：
├── 首页：进入训练
├── 方案页：建立信任
├── 训练页：执行指导
├── 反馈页：记录数据
└── 进步页：感知价值（核心）
```

## 六、设计语言

### 6.1 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | #F7F4F0 | 温暖浅色 |
| 卡片 | #FFFFFF | 清晰分层 |
| 主色 | #48BB31 | 正向绿 |
| 文字主 | #222222 | 深色易读 |
| 文字次 | #666666 | 辅助说明 |
| 温暖提示 | #FFF8F0 | 暖黄底色 |
| 成功 | #E8F5E9 | 浅绿底 |

### 6.2 字体

- 主字体：系统默认（苹方/思源/微软雅黑）
- 标题：16-18px，加粗
- 正文：14-15px，常规
- 辅助：12-13px，浅色

### 6.3 间距

- 页面边距：20px
- 卡片间距：16px
- 元素间距：12px
- 紧凑间距：8px

### 6.4 圆角

- 大圆角：20-28px（页面级卡片）
- 中圆角：12-16px（按钮、输入框）
- 小圆角：8px（标签、徽章）

### 6.5 阴影

- 卡片阴影：0 2px 12px rgba(0,0,0,0.06)
- 按钮阴影：0 4px 12px rgba(72,187,49,0.2)

## 七、状态设计

### 7.1 按钮状态

| 状态 | 样式 |
|------|------|
| 默认 | 绿色背景，白色文字 |
| 按下 | 颜色加深10% |
| 禁用 | 灰色背景，浅色文字 |

### 7.2 卡片状态

| 场景 | 卡片 | badge |
|------|------|-------|
| 进行中 | 白底 | 🔥 绿色底 |
| 已完成 | 淡绿底 | ✅ 蓝色底 |

### 7.3 输入状态

| 状态 | 样式 |
|------|------|
| 空状态 | 浅灰边框 |
| 聚焦 | 绿色边框 |
| 有内容 | 深色边框 |

## 八、技术实现要点

### 8.1 数据持久化

- localStorage存储anonymous UUID
- 训练记录本地保存
- 进度实时更新

### 8.2 微感知文案

- 硬编码50条模板
- 模板+变量组合
- 每日反馈后展示

### 8.3 里程碑触发

- 每日反馈提交后检查 day % 7 == 0
- 达成时展示里程碑卡片

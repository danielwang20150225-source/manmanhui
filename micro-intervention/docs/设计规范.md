## 一、设计尺寸规范

| 元素 | 规格 |
|------|------|
| 手机框架 | 最大宽度480px，高度100vh，圆角36px |
| 状态栏 | 高度44px，深色(#1A1A1A) |
| 底部导航 | 高度64px，底部固定，圆角0 |
| 页面内边距 | 水平16px，垂直20px |
| 卡片间距 | 卡片之间12px |
| 圆角系统 | 小:12px，中:16px，大:24px，超大:36px |

## 二、色彩系统

| 类型 | 色值 | 用途 |
|------|------|------|
| 主色 | #7CB342 | 按钮、进度条、强调 |
| 主色深 | #689F38 | 按钮按下态 |
| 主色浅 | #E8F5E9 | 背景、选中态 |
| 背景 | #F7F4F0 | 页面背景（暖白） |
| 卡片 | #FFFFFF | 卡片背景 |

| 文字主 | #2D2D2D | 主要文字 |
| 文字次 | #6B6B6B | 次要说明 |
| 文字弱 | #ADADAD | 占位符、辅助说明 |

## 五、交互规范

### 动画时长

| 交互 | 时长 | 缓动 |
|------|------|------|
| 点击反馈 | 150ms | ease-out |
| 页面切换 | 300ms | ease-in-out |
| 展开收起 | 300ms | ease |
| 加载动画 | 1.5s | linear (循环) |
| 淡入 | 400ms | ease-out |
| 滑动删除 | 200ms | ease |

### 反馈效果

| 反馈 | 效果 |
|------|------|
| 点击按下 | transform: scale(0.97)，阴影消失 |
| 卡片进入 | opacity 0→1，translateY(16px→0) |

## 六、CSS 变量定义

```css
:root {
  /* 尺寸 */
  --phone-width: 480px;
  --status-bar-height: 44px;
  --tab-bar-height: 64px;
  --page-padding: 16px;
  --card-gap: 12px;

  /* 圆角 */
  --radius-sm: 12px;
  --radius-md: 16px;
  --radius-lg: 24px;
  --radius-xl: 36px;

  /* 色彩 */
  --accent: #7CB342;
  --accent-dark: #689F38;
  --accent-light: #E8F5E9;
  --bg: #F7F4F0;
  --card: #FFFFFF;
  --border: #E8E4E0;
  --text-primary: #2D2D2D;
  --text-secondary: #6B6B6B;
  --text-hint: #ADADAD;
  --success: #4CAF50;
  --warning: #F5A623;
  --error: #E57373;

  /* 阴影 */
  --shadow: 0 2px 12px rgba(0,0,0,0.06);

  /* 字体 */
  --font-family: -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif;
}
```

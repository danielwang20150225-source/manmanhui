# 微干预V2.0原型设计规范

> 版本：V2.0
> 日期：2026-05-13
> 与Daniel通过原型图深度讨论确定

---

## 一、设计背景与目标

2026-05-13，与Daniel通过原型图（微干预V2.0.png）深度讨论微干预Demo版重构方案。目标是基于原型设计，将现有的Demo版本按新设计进行重构。

## 二、设计原则

- 一个页面只做一个核心任务
- 极简化设计，减少冗余元素
- 游戏化激励，增强用户坚持动力
- 移动端优先，单手操作友好

## 三、页面结构总览

### 3.1 底部导航（三Tab）

| Tab名称 | 图标 | 功能说明 |
|---------|------|----------|
| 家长干预 | 👨‍👩‍👦 | 任务列表+输入 |
| 孩子练习 | 🧒 | 占位页（未来桌面干预模块） |
| 我的 | 👤 | 孩子档案+设置 |

### 3.2 页面清单

| 序号 | 页面 | 文件 | 核心功能 |
|------|------|------|----------|
| 1 | 首页/情境输入 | input.html | 输入孩子行为问题，生成任务 |
| 2 | 加载页 | loading.html | AI生成中，显示处理步骤 |
| 3 | 任务确认页 | package.html | 查看生成的任务，进入训练 |
| 4 | 任务详情页 | task-detail.html | 当天训练步骤、目标、观察重点 |

| 8 | 训练记录页 | history.html | 历史训练数据统计 |
| 9 | 我的 | me.html | 孩子档案、设置入口 |

## 四、各页面详细设计

### 4.1 首页/情境输入（input.html）

| 元素 | 设计说明 |
|------|----------|
| 顶部导航 | 无，极简化 |
| 输入框 | 多行文本框 |
| 快捷标签 | 6个标签，2×3宫格布局，点击直接生成 |
| 标签内容 | 穿衣抗拒👕、外出困难🚗、呼名不应👂、社交回避👥、情绪崩溃😢、感官敏感🤯 |
| 语音输入 | 有语音输入按钮 |
| 按钮文案 | 生成干预任务 |
| 卖点文案 | 一句话精简（参考原型1） |

### 4.2 加载页（loading.html）

| 元素 | 设计说明 |
|------|----------|
| 步骤文案 | 四步：理解情境→分析问题→匹配任务包→生成方案 |
| 超时处理 | 超时提示+重试按钮 |

### 4.3 任务确认页（package.html）

| 元素 | 设计说明 |
|------|----------|
| 顶部导航 | 无或仅返回箭头 |
| 成功提示 | 任务已生成+勾选图标 |
| 任务卡片 | 任务名称、目标、进度条、状态标签 |
| 进度展示 | 圆点（7天）+文字（第X天/共7天） |
| 连续天数 | 不需要，简化页面 |
| 主按钮 | 开始训练 |
| 副按钮 | 返回任务列表 |

### 4.4 任务详情页（task-detail.html）

| 元素 | 设计说明 |
|------|----------|
| 布局 | 任务名称→今日目标→训练步骤→观察重点→反馈按钮 |
| 步骤展示 | 卡片列表，每个步骤一个卡片，带序号和图标 |
| 观察重点 | 提示气泡，浅色背景+灯泡图标 |
| 进度保存 | 不需要自动保存，仅展示内容 |

### 4.5 反馈页（feedback.html）

| 元素 | 设计说明 |
|------|----------|
| 训练次数 | 计数器（+/-按钮） |
| 情感反馈 | 三个表情（😊😐😣） |
| 文字备注 | 可选输入 |
| 提交后跳转 | 进入完成/奖励页 |

### 4.6 完成页（progress.html）

| 元素 | 设计说明 |
|------|----------|
| 顶部装饰 | 星光/庆祝元素 |
| 情感标题 | 今天的进步 |
| 鼓励语 | 一句话激励文案 |
| 游戏化激励 | 连续X天徽章 |
| 明日预告 | 不需要 |
| 主按钮 | 再来一次 |
| 副按钮 | 返回任务列表 |
| 底部导航 | 无，极简化 |

### 4.7 任务列表页（index.html）

#### 任务卡片布局

| 区域 | 内容 |
|------|------|
| 顶部 | 任务名称+目标说明 |
| 右上角 | 删除按钮（滑动显示） |
| 中部 | 最新反馈（表情+文字，可隐藏）+更多链接 |
| 下部 | 进度条+进度文字（第X天/共X天） |
| 底部 | 进入任务按钮 |

#### 空状态设计

| 场景 | 表现 |
|------|------|
| 首次使用 | 用户不会触达此页面（无入口） |
| 全部删除 | 显示引导文案+快捷入口添加新情境 |
| 全部完成 | 已完成任务折叠显示+可展开+引导入口 |

| 元素 | 设计说明 |
|------|----------|
| 进行中列表 | 任务卡片，显示进行中的任务 |
| 已完成入口 | 折叠式，默认折叠，顶部显示"查看已完成（N个）" |
| 新建入口 | 列表末尾卡片型，虚线边框+大+图标，文案：添加新情境 |
| 额度逻辑 | 最多同时进行3个任务 |
| 删除功能 | 滑动卡片露出红色删除按钮 |

### 4.8 训练记录页（history.html）

| 元素 | 设计说明 |
|------|----------|
| 入口 | 任务卡片更多入口下钻 |
| 顶部导航 | 返回键+页面标题训练记录 |
| 汇总区 | 训练科目+总训练次数 |
| 统计区 | 三栏卡片（顺利/一般/有点难） |
| 历史列表 | 按月分组，每条记录卡片 |

### 4.9 我的页面（me.html）

| 元素 | 设计说明 |
|------|----------|
| 孩子档案卡片 | 头像（默认男孩/女孩选择）+姓名（默认值）+年龄+性别 |
| 设置区 | 关于/隐私协议（合二为一） |

#### 孩子档案默认值

| 字段 | 默认值 |
|------|--------|
| 头像 | 不编辑，仅男孩/女孩选择，默认为男孩 |
| 姓名 | 有初始默认值 |
| 年龄 | 默认为空 |
| 性别 | 默认为空 |

### 4.10 孩子练习Tab

目前为占位页，显示"正在开发中"。未来是桌面干预模块。

## 五、页面跳转关系

```
首页→加载页→任务确认页→任务详情页→反馈页→完成页
                                               ↓
                                    （再来一次→任务详情页）
                                               或
                                    （返回任务列表）

任务列表页→任务详情页（进入任务）
任务列表页→训练记录页（更多入口）

我的→孩子档案编辑页→返回
我的→关于/隐私协议页→返回
```

## 六、技术实现要点

- **路由守卫**：不需要隐私协议弹窗，用户直接使用即代表同意
- **数据存储**：localStorage+sessionStorage
- **API调用**：后端FastAPI在localhost:8000
- **设计风格**：温暖家庭风，绿色(#7CB342)主色调

## 七、设计规范系统

### 9.1 设计尺寸规范

| 元素 | 规格 |
|------|------|
| 手机框架 | 最大宽度480px，高度100vh，圆角36px |
| 状态栏 | 高度44px，深色(#1A1A1A) |
| 底部导航 | 高度64px，底部固定 |
| 页面内边距 | 水平16px，垂直20px |
| 卡片间距 | 12px |
| 圆角系统 | 小:12px，中:16px，大:24px，超大:36px |

### 9.2 色彩系统

| 类型 | 色值 | 用途 |
|------|------|------|
| 主色 | #7CB342 | 按钮、进度条、强调 |
| 主色深 | #689F38 | 按钮按下态 |
| 主色浅 | #E8F5E9 | 背景、选中态 |
| 背景 | #F7F4F0 | 页面背景（暖白） |
| 卡片 | #FFFFFF | 卡片背景 |
| 边框 | #E8E4E0 | 卡片边框、分隔线 |
| 文字主 | #2D2D2D | 主要文字 |
| 文字次 | #6B6B6B | 次要说明 |
| 文字弱 | #ADADAD | 占位符、辅助说明 |
| 成功 | #4CAF50 | 顺利标签 |
| 警告 | #F5A623 | 一般标签 |
| 错误 | #E57373 | 有点难标签、删除 |

### 9.3 字体系统

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 24px | 700 | 页面大标题 |
| H2 | 20px | 700 | 卡片标题 |
| H3 | 17px | 600 | 区块标题 |
| 正文 | 15px | 400 | 主要内容 |
| 辅助 | 14px | 400 | 说明文字 |
| 小字 | 13px | 400 | 标签，元信息 |
| 最小 | 12px | 400/600 | 徽章、提示 |

**字体**: -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif

### 9.5 交互规范

#### 动画时长

| 交互 | 时长 | 缓动 |
|------|------|------|
| 点击反馈 | 150ms | ease-out |
| 页面切换 | 300ms | ease-in-out |
| 展开收起 | 300ms | ease |
| 加载动画 | 1.5s | linear |
| 淡入 | 400ms | ease-out |
| 滑动删除 | 200ms | ease |

| 反馈 | 效果 |
|------|------|
| 点击按下 | transform: scale(0.97)，阴影消失 |
| 卡片进入 | opacity 0→1，translateY(16px→0) |
